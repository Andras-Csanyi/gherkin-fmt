# gherkin-fmt

A CLI tool to format Gherkin files.
The motivation to create and run this project is having a Gherkin file formatter
in NeoVim.

## It is good to know

This formatter is generated by AI. I wrote the Gherkin files under the `spec/`
directory and Grok did the code generation.

The formatter at this point satisfies my needs which may be little for your needs.

## Features

- Format Gherkin feature files
- Support for `Feature`, `Scenario`, `Scenario Outline` and `Background`
- formats horizontal and vertical tables
- Debug mode for tracking formatting rules

## Installation

The below requires Rust being installed on your machine.

```bash
cargo install gherkin-fmt
```

## Usage

```bash
gherkinfmt input.feature [output.feature] --debug
```

## Conform integration

The formatter can read from `STDIN` like below:

```bash
echo "feature: something" | gherkinfmt -- -
```

Where the result is going to be:

```bash
Feature: something
```
